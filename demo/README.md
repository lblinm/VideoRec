# 运行

1. 确保具有环境：`python >= 3.7 (perfect 3.9/3.10)`
2. 安装依赖：`pip install -r demo/requirements.txt`
3. 运行`main.py`文件：`python demo/main.py`

> release 发行版: 下载解压后双击`VideoTool.exe`

# 功能

1. 模拟用户行为，生成评分矩阵
2. SVD、Content Based、Item based CF 等多种推荐算法
3. 视频聚类
4. 用户聚类
5. 模拟视频历史播放量
6. 预测视频未来播放量
